### 基于Vue的前端页面功能

  1.用户注册/登录/修改密码


2.登录页面使用vue-particles插件画粒子特效


3.展示实验基本信息，基本状态，实验参数，对蓄电池充放电机的控制操作


4.使用Echarts画电压，电流最新十条数据的变化趋势，并标定最大值，最小值，平均值


5.实现对蓄电池充放电机的工艺号设置操作


6.查看实验历史数据
